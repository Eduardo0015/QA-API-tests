ID: CT001
Title: Validar retorno de usuário existente
Endpoint: GET /api/users/2
Preconditions: 
 - API ReqRes acessível
 - Usuário com ID 2 existente na base pública
Steps
1. Enviar requisição GET https://reqres.in/api/users/2
2. Capturar resposta da API
Expected Result:
 - Status code 200 OK
 = Body contém campo data.id=2
 - Body contém campo data.email não vazio
 - Content-type = application/json
 - Tempo de resposta <2000ms
