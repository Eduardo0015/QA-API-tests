ID: CT005
Title: Criando usuário inválido
Endpoint POST api/users
Preconditions: 
 - API ReqRes disponível
 - Usuário não existe na base de dados pública
Steps: 
1. Enviar POST https://reqres.in/api/users
2. Body JSON:
 {
    "name": "Eduardo",
    "username": ,
    "email": "test@tst.com "
}
3. Capturar resposta
Expected Result:
 - Status Code 400 Bad Request
 - Tempo de resposta < 2000ms
