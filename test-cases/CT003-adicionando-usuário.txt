ID: CT003
Title: Adicionando novo usuário válido
Endpoint: POST /api/users
Preconditions:
 - API ReqRes acessível
Steps:
1. Enviar POST https://reqres.in/api/users
2. Body JSON:
   {
     "name": "Eduardo",
     "job": "QA"
   }
3. Capturar resposta
Expected Result:
- Status code 201 Created
- Body contém campo "name" = "Eduardo"
- Body contém campo "job" = "QA"
- Body contém campo "id" gerado
- Body contém campo "createdAt"
- Content-Type = application/json
- Tempo de resposta < 2000 ms
