# Testes de API – DummyJSON

## Visão Geral

Projeto de testes de API realizado utilizando a DummyJSON, uma API pública utilizada para simular dados de aplicações web.

O objetivo do projeto foi praticar validação de endpoints REST, estrutura de respostas JSON e automação básica de testes.

## API Utilizada

DummyJSON – API pública para simulação de dados de produtos, usuários e autenticação.

## Escopo dos Testes

Foram realizados testes nos seguintes endpoints:

* Produtos
* Usuários
* Autenticação

## Ferramentas Utilizadas

* Postman
* Cypress
* JavaScript
* Git e GitHub

## Objetivo do Projeto

Praticar conceitos fundamentais de testes de API:

* Validação de status codes
* Estrutura de respostas JSON
* Testes positivos
* Automação básica de requisições
